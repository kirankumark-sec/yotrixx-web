## 📌 Overview
A modern, highly interactive landing page for **Yotrixx**, a next-generation learning platform for SketchUp, Lumion, Architecture, Interior Design, and Construction. This front-end project is designed to capture leads and drive course enrollments through a seamlessly embedded Google Form, wrapped in a premium, dark-themed UI.

## ✨ Key Features
- **Modern UI/UX:** Sleek dark theme with neon-mint accents, frosted glass effects (backdrop-filter), and a noise grain overlay for texture.
- **Interactive Canvas Hero:** Custom particle field background with proximity connection lines powered by the HTML5 `<canvas>` API.
- **Custom Cursor Interaction:** Animated dot and ring cursor that reacts to hover states on interactive elements.
- **Performance Optimized:** Uses `IntersectionObserver` for smooth, lightweight scroll reveals and dynamic counter animations.
- **Seamless Registration:** Integrated Google Form wrapper styled to match the native environment.
- **Fully Responsive:** CSS Grid and Flexbox layouts ensure a perfect experience across desktop, tablet, and mobile devices.
- **Quick Connect:** Floating WhatsApp action button for instant user support.

## 🚀 Tech Stack
- **Markup:** HTML5 (Semantic Structure)
- **Styling:** CSS3 (CSS Variables, Flexbox, Grid, Keyframe Animations)
- **Scripting:** Vanilla JavaScript (ES6+, DOM Manipulation, Canvas API)
- **Typography:** Google Fonts (Syne, Plus Jakarta Sans, DM Mono)
